# Eatsy

Eatsy is a recipe recommendation application developed as an MCA mini project. The system recommends recipes based on ingredients provided by the user through text input or image upload.

The project uses the InceptionV3 deep learning model for ingredient recognition and TensorFlow Lite for Android integration.

## Features

- Ingredient detection using images
- Recipe recommendation based on available ingredients
- Text and image input support
- Android application interface
- TensorFlow Lite model integration
- Real-time ingredient recognition

## Technologies Used

- Python
- TensorFlow
- TensorFlow Lite
- InceptionV3
- Android Studio
- Java
- OpenCV

## Dataset Used

- Fruits-360 Dataset
- Indian Food Recipe Dataset
- Non-Vegetarian Foods Dataset

## Model Performance

- Validation Accuracy : 95.39%
- Precision : 93.99%
- Recall : 92.58%
- F1 Score : 92.68%

## Future Scope

- Multi-ingredient detection
- Nutritional analysis
- Voice-based ingredient input
